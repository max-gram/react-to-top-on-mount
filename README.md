# react-to-top-on-mount
Tiny React component to scroll window to top on componentDidMount

## Usage

```
import ToTopOnMount from 'react-to-top-on-mount'

class SomeName extends Component {
  render() {
    <div>
      <ToTopOnMount />
      <div>
        Some content
      </div>
    </div>
  }
}

```
